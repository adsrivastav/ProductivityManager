# Personal Productivity Manager

This project is given to me as a company assignment.

I tried to complete all requirements one by one and learned many things while doing this assignment.

## Tech Stack

Frontend:

* React.js
* Tailwind CSS

Backend:

* Node.js
* Express.js

Database:

* MongoDB (Mongoose)

Authentication:

* JWT (Access Token & Refresh Token)

## Features

### Authentication

* User Register
* User Login
* Logout
* Role based access (User / Admin)

### Expense Manager

* Add expense
* Edit expense
* Delete expense
* View expense list
* Filter by category, type and date

### Task Manager

* Add task
* Edit task
* Delete task
* Task status (pending, in-progress, completed)

### Dashboard

* Total income
* Total expense
* Total tasks
* Completed tasks

## Folder Structure (Basic)

Frontend

* pages/
* components/


Backend

* controllers/
* routes/
* models/
* middleware/

## How to Run Project

### Backend

1. Go to backend folder
2. Run `npm install`
3. Create `.env` file
4. Run `npm start`

### Frontend

1. Go to frontend folder
2. Run `npm install`
3. Run `npm run dev`

## Environment Variables

Backend `.env`

* PORT
* MONGO_URI
* JWT_SECRET
